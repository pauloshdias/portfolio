This project is meant to produce a design system for an online store that sells only this marvelous item.

The spacing follows the 8 point grid with the exception of a minimum 4 pixels. This grid allows for an easier scalling across platforms.

The goal of this design was meant to be as easy and as simples to use as possible. For that reason, and because de product allowed it the product page was embedded in the main homepage.

To a cardistry affectionate, the most important is to see the package, the card’s back, the court cards and an example of a numbered card. On top of these, it’s important to offer a description of the deck’s concept and a word about the designer.

This design solution allows for a huge reduction of pages and clicks merging the address and payment inputs as well as the card confirmation pages.

This way the user can deal with all the requirements all on one page. Note for the payment methods wich each option will open a different set of input requirements underneath the chosen option.

This not only reduces the number of processes the user has to go through as well as reduces the number of pages to be developed.
